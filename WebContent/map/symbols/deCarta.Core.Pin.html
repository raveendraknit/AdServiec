Class deCarta.Core.Pin

A Pin is used to graphically represent a position on the map. It can have a custom image, and when clicked it can display custom text. Add the pin to a map overlay using the deCarta.Core.MapOverlay:addObject() method. Pin extends deCarta.Core.OverlayObject.

Class Detail
deCarta.Core.Pin(options)
An OverlayObject used to display a pin on a MapOverlay
Parameters:
options
Options. May contain one or more of the following:
  • (string) text: text diplayed on the pin info window - optional, default=''
  • {HTMLDOMElement} textElement: if this option is set, this will be an element that overrides the standard pin label
  • deCarta.Core.Position position: the geographic position at which to place the pin, required
  • (HTMLDOMElement) image: a DOM image element representing the pin
  • (string) imageSrc: a string containing either the URI to an image or a base64 representation of the image
  • (int) xOffset: Number of pixels to shift the pin to the left (set to half of the image width to center the image over the position on the map), optional, default=0
  • (int) yOffset: Number of pixels to shift the pin up (set to the full height of the image to place the image above the position on the map), optional, default=0
  • (function) onClick: a callback function that will be notified if the pin is clicked
  • (function) onLabelClick: a callback that will be notified if the label is clicked
  • (function) onShowText: this function will be invoked when the pin's infowindow is shown
  • (function) onHideText: this function will be invoked when the pin's infowindow is hidden
  • (function) onBeforeShowText: this function will be invoked before the pin's infowindow is shown - returning false will prevent the infowindow from being displayed.
  • (function) onBeforeHideText: this function will be invoked before the pin's infowindow is hidden - returning false will prevent the window from closing
  • (function) onImageLoad: invoked when the pin's image completes loading
  • (string) cssClass: this css class will be added to the pin. Can be used for styling
  • (deCarta.Core.InfoWindow) infoWindow: If you require the pin to have it's own infoWindow, instead of using the global object, you can pass an instance of {deCarta.Core.InfoWindow} in this property
  • (boolean) textVisible: Controls visibility of pin text. Default=false (click pin to toggle to true)
  • (boolean) opaque: events will stop at the pin, and not propagate down to the map
