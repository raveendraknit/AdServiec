Class deCarta.Core.EventableObject

the EventableObject class is a base class which can be extended to create objects which respond to user input. All OverlayObjects and the Map objet extend this class. This class installs all necessary event listeners, and also creates all shortcut event methods on the object (ex, object.onclick()) The class provides a set of standard methods for event handling, which the subclass will have to override if it needs to provide specific functionality.
