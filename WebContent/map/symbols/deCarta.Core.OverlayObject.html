Class deCarta.Core.OverlayObject

OverlayObject is a base class that can be extended to create objects suitable for display on a deCarta.Core.MapOverlay. Pins deCarta.Core.Pin, Polylines deCarta.Core.Polyline, Circles deCarta.Core.Circle, Images deCarta.Core.Image, Polygons deCarta.Core.Polygon all extend this class.
