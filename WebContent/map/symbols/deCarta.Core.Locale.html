Class deCarta.Core.Locale

A Locale object represents a specific geographical, political, or cultural region. An operation that requires a Locale to perform its task is called locale-sensitive and uses the Locale to tailor information for the user. Currently, Locales can be attached to a deCarta.Core.FreeFormAddress object to aid in producing better results with a deCarta.Core.Geocoder:geocode().

These pairs of language can be used to create a valid locale for use with the Geocoder.
